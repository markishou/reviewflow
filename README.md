# ReviewFlow

AI Code Review Bottleneck Manager — intelligent PR routing and prioritization for teams using AI coding assistants.

## About

ReviewFlow is a code review orchestration platform that sits between your Git provider and your team, intelligently managing the flow of pull requests to maximize review throughput.

## Tech Stack

- **Frontend:** React 18 + TypeScript + Tailwind CSS
- **Backend:** AWS Lambda (Node.js 24.x)
- **API:** AWS API Gateway
- **Auth:** AWS Cognito
- **Database:** Neon (Serverless PostgreSQL)
- **Hosting:** S3 + CloudFront
